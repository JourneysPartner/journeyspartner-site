About

私たちについて

株式会社JourneysPartnerは、兵庫県加古郡稲美町に拠点を置くBtoB支援会社です。 中小企業・個人事業主の皆さまに対し、記帳代行・会計代行・経理代行といったバックオフィス支援から、 集客支援・広告運用、経営／人材／労務コンサルティングまで、事業運営に必要な実務を幅広くサポートします。

また、税務申告等の専門領域については、提携先である毛利順活税理士事務所と連携し、 役割を明確に分けたうえで安心してご利用いただける体制を整えています。

Services

主要サービス

バックオフィスから集客・経営支援まで、実務に直結する8つのサービスを提供しています。

  • 記帳代行

    日々の取引記録を正確に処理し、月次試算表の把握を支援します。

  • 会計代行

    会計処理全般を代行し、経営者が本業に集中できる環境を整えます。

  • 経理代行

    請求・支払・振込業務など、経理実務をまとめて支援します。

  • 集客支援

    Web・リアル両面から、事業特性に合った集客施策を検討・実行します。

  • 広告運用

    予算と目的に応じた広告の設計・運用・改善を継続的に行います。

  • 経営コンサルティング

    数値と現場の両面から、経営課題の整理と意思決定を支援します。

  • 人材コンサルティング

    採用・組織づくり・定着に関する課題へ実務目線で伴走します。

  • 労務コンサルティング

    労働環境・就業ルールの整備に関するご相談に対応します。

※ 税務申告等の税務に関する専門対応は、提携先の毛利順活税理士事務所が担当します。

Google Ads API Usage

Google Ads API の利用目的

株式会社JourneysPartnerは、自社および提携サービスの集客を目的として、 Google Ads API を社内の内部ツールとして利用しています。

本APIを用いて第三者向けに販売・提供するツールやサービスの開発は行っておらず、 外部向け販売ツールではありません。 あくまで自社業務における広告運用・改善のために利用しています。
